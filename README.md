# pick-omit

[![npm version](https://img.shields.io/npm/v/pick-omit.svg)](https://www.npmjs.com/package/pick-omit)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Object pick, omit, and deep merge utilities

## Installation

```bash
npm install pick-omit
```

## Functions

- `pick` — Pick specified keys from object
- `omit` — Omit specified keys from object
- `deepMerge` — Deep merge two objects
